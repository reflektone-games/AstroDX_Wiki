## 顶层属性

| 属性 | 类型 | 默认值 | 描述 |
|----------|------|---------|-------------|
| `name` | `string` | 无 | 皮肤的名称。此名称会显示在皮肤选择菜单中。 |
| `designer` | `string` | 无 | 皮肤设计者的名字。 |
| `ring` | `SpriteDefinition` | `{ "path": "ring.png", "ppu": 115 }` | 覆盖游玩区域的环状图像。 |


<Callout 类型="info">

    如果你未在 JSON 文件中声明这些属性，它们将使用默认值。
    无效路径将被忽略，不会在游戏中作为选项显示。

</Callout>

## 颜色

这些属性定义了音符后方引导标记的着色颜色。颜色以十六进制形式表示（例如 `"#FF5733"` 或带透明度的 `"#FF5733FF"`）。

| 属性 | 类型 | 默认值 |
|----------|------|---------|
| `tap_color` | `Color` | 无 |
| `each_color` | `Color` | 无 |
| `break_color` | `Color` | 无 |
| `mine_color` | `Color` | 无 |
| `slide_color` | `Color` | 无 |

## SpriteDefinition 类型

每个图像属性都使用 `SpriteDefinition` 类型，包含以下字段:

| 属性 | 类型 | 默认值 |描述 |
|----------|------|---------|-------------|
| `path` | `string` | 无 | 图像文件在皮肤文件夹内的相对路径。 |
| `ppu` | `float` | `200` | 游戏内每单位像素数。值越小，图像显示越大。值为 100 表示每单位包含100个像素点。 |
| `pivot_x` | `float` | `0` | 精灵的水平轴心点，值从 0（左）到 1（右）。值为 0.5 时水平轴心居中。 |
| `pivot_y` | `float` | `0` | 精灵的垂直轴心点，值从 0（下）到 1（上）。值为 0.5 时垂直轴心居中。 |
| `top` | `float` | `0` | 用于九宫格图像的上边框，以像素为单位从顶部边缘测量。 |
| `bottom` | `float` | `0` | 用于九宫格图像的下边框，以像素为单位从底部边缘测量。 |

<Callout type="info">

    支持的图像格式与 Unity 的 `Texture2D.LoadImage` 方法所支持的格式相同，包括 PNG、JPEG 及其他常见格式。

</Callout>
