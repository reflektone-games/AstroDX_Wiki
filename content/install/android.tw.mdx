### **Import via ADX archive** **(Recommended, Beta 2 Patch 4 and later)**

This is the primary way to install levels, and itâ€™s incredibly easy to get started.

<Steps>

<Step>
  ### Rename your zip file
  Any given `.zip` file of a level or multiple levels can be **renamed** into a `.adx` file.
</Step>

<Step>
  ### Open with... > AstroDX
  On supported devices, tap on the file in any file manager, and select `Open withâ€¦ > AstroDX`
</Step>

<Step>
  ### Profit
  AstroDX will automatically install levels for you [depending on what the archive contains](https://astrodx.notion.site/ADX-Archive-Details-a8346048819a40b18f0a6c014c19b0b3?pvs=74).
</Step>

</Steps>

<Callout title="On some devices, (for example Xiaomi or some Samsung devices), your file manager may not offfer the option to open with AstroDX." type="warn">

Try one of these options:
[**ZArchiver**](https://play.google.com/store/apps/details?id=ru.zdevs.zarchiver&hl=en) /
[**Android SAF**](https://play.google.com/store/apps/details?id=com.marc.files&pcampaignid=web_share)

For these file managers, if you don't see the `Open with...` prompt, you may be able to access it using the **Share function** instead.
</Callout>

Rename any `.adx` file back into `.zip` to extract it as usual.

<Callout title="Android's limitation: .adx files cannot be larger than 2GB">

You will encounter errors when attempting to install .adx archives that are larger than 2GB in size. <br />This is a hard Android requirement, and we **cannot fix this.**<br />
We **plan to address this in the future**.
